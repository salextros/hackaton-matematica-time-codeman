# TIME CodeMan - Misión Matemática contra Olvidín

Página web infantil gamificada para la Actividad 1: Hackatón pedagógica virtual.

## Incluye

- Imagen/banner de TIME CodeMan.
- Mascota CodeMan 10.0.
- Misión de multiplicación básica con Kahoot.
- Video del villano Olvidín.
- Actividad de la tabla del 7 en LearningApps.
- Canción de la tabla del 8 en video y audio de apoyo.
- Batalla final en Genially.
- Conclusiones y referencias.
- Espacio para video pitch académico.
- Barra de progreso por misiones.

## Videos

La página ya reconoce estos archivos dentro de la carpeta `videos`:

```text
videos/villano-olvidin.mp4
videos/cancion-tabla-8.mp4
videos/pitch-academico.mp4
```

El archivo `pitch-academico.mp4` es opcional. Si no existe, la página mostrará un aviso de video pendiente.

## Publicar en GitHub Pages

```bash
git init
git branch -M main
git add .
git commit -m "chore: subir pagina time codeman hackaton"
git remote add origin https://github.com/TU_USUARIO/NOMBRE_DEL_REPOSITORIO.git
git push -u origin main
```

Luego en GitHub:

```text
Settings → Pages → Deploy from a branch → main → /root → Save
```
